# AceSentinel

AceSentinel is a Cloud Security Posture Management (CSPM) platform that scans AWS environments for misconfigurations and security risks.

## Features
- Detects public S3 buckets
- Identifies buckets without encryption
- Flags insecure IAM configurations
- Scans EC2 security groups for open ports
- Generates risk-based security reports

## Tech Stack
- Python (boto3, FastAPI)
- AWS
- JSON reporting

## Roadmap
- Web dashboard
- AI-generated remediation insights
- Compliance mapping (NIST, CIS)
